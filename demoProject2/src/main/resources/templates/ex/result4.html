타임리프 예제4(조건문)

th:if / th:unless 

    th:if = "조건식"
    - 조건식이 true인 경우에 수행

    th:unless = "조건식" == else
    - 조건식이 false인 경우에 수행

std 없음

std.toString 출력

조건식에 작성된 변수에 값이 존재하는지 확인하는 방법

th:if = "${변수명|key}"
-> 변수에 값이 있으면 true, 없으면 false

삼항 연산자 

    (조건식 ? : true인경우 : false인 경우)

Elvis 연산자 ( ?: )

    [작성법]
    값 ?: 값이 없을 때

    - 값이 존재하는 경우      -> 값 출력
    - 값이 존재하지 않은 경우 -> 값이 없을 떄 출력

No-Operation 연산자 ( 값 ?: _)

    - 값이 존재하지 않을경우
       태그 사이에 작성된 내용(content 영역)을 
                                        출력하는 연산자
